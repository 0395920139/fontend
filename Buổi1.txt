@ Khái niệm cơ bản 

- HTML là gì ?
	=> là chữ viết tắt của Hypertext Markup languge, Nó giúp người tạo và cấu trúc các thành phần trong trang web
	hoặc ứng dụng,phân chia các đoạn văn,heading,links,blockquoetes,vv
	
- CSS là gì ?
	=> Là ngôn ngữ tạp phong cách(style) cho trang web - cascading style sheet language. Nó dùng để tạo phong cách
	và định kiểu cho những yếu tố đc viết dưới dạng ngôn ngữ đánh dấu như html
